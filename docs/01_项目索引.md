## 🚀 系统启动命令速查

### OrangePi 5 Ultra（生产环境）

```bash
cd ~/Code/Project/BOTDOG/BotDog

# 1. 激活虚拟环境
source .venv/bin/activate

# 2. 启动视频流水线（H.265 → H.264 → MediaMTX WHEP）
bash scripts/run-pipeline.sh

# 3. 启动后端
bash scripts/start_backend.sh

# 4. 浏览器访问
# http://<OrangePi_IP>:8000
```
