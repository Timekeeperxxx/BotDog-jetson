# 纯视觉要害区域自动驱离 — 分阶段实施计划

> 基于 [16_纯视觉要害区域自动驱离实现路径.md](16_纯视觉要害区域自动驱离实现路径.md)
> 和 [17_BotDog_视觉锚点防区驱离_需求文档与实现思路.md](17_BotDog_视觉锚点防区驱离_需求文档与实现思路.md) 制定。

## 总体改造思路

将 `GuardMissionService` 从当前的 **"内嵌 HSV 检测 + 静态 bbox + 定时盲退"** 改造为 **"独立动态区域检测模块 + 视觉伺服闭环推进/返航"**。

**核心原则：先做稳区域识别，再接前进，再接返航。每一步做完都能独立验收，不跑通上一步绝不进下一步。**

---

## 当前代码现状

| 模块 | 文件 | 现状 | 问题 |
|------|------|------|------|
| 区域检测 | `guard_mission_service.py` 内嵌 `_detect_zone()` | 用 HSV(10-40) 暖色分割 + minAreaRect | 无黑边验证、无稳定保持、检测逻辑和业务逻辑耦合 |
| 入侵判定 | `guard_mission_service.py` 内 `_is_foot_in_zone()` | 脚点进 polygon 已实现 | 基本可用，小幅优化即可 |
| 前进控制 | `guard_mission_service.py` 内 `_on_advancing()` | bbox center 做粗偏航 + 面积比做停止 | 未调用 `VisualServoController`，逻辑直接嵌在状态机里 |
| 返航控制 | `guard_mission_service.py` 内 `_on_returning()` | **固定 5 秒盲退** | 完全不是视觉闭环 |
| 视觉伺服 | `visual_servo_controller.py` | 方法已定义 | **未被主流程调用**，形同虚设 |
| 锚点跟踪 | `visual_anchor_tracker.py` | OpenCV TrackerMIL | **已弃用**，新方案靠帧级 HSV 检测替代 |
| 状态机 | `guard_mission_types.py` | 6 态 | `LOCK_ANCHOR` / `FAULT` 在当前流程中未使用 |

## 阶段 1：稳定识别区域

### 目标

无论有没有人、摄像头是否轻微抖动、背景里有没有杂物，系统都能稳定输出黄色+黑边区域的 polygon / bbox / center / area / angle / quality。

**这一步先完全不要让狗动。**

### 新增文件

#### `backend/yellow_zone_detector.py`（新增）

独立模块，与 `GuardMissionService` 解耦，单独负责区域检测。

**输出数据结构**：

```python
@dataclass
class ZoneDetection:
    polygon: np.ndarray           # shape (4, 2)，四边形顶点
    bbox: Tuple[int, int, int, int]  # (x, y, w, h)
    center: Tuple[int, int]       # 中心点 (cx, cy)
    area: float                   # 像素面积
    angle: float                  # 旋转角度（度）
    quality: float                # 0.0-1.0 综合质量分
```

**内部检测管线（第一版）**：

```
1. BGR → HSV
2. 阈值分割黄色区域（H:20-35，比当前 10-40 暖色范围更聚焦纯黄）
3. 形态学闭运算填洞 + 开运算去噪
4. 找外轮廓
5. 按面积 / 长宽比 / 位置过滤候选
6. 用 minAreaRect 拟合四边形（第一版不强求 approxPolyDP）
7. 黑边验证：在四边形外扩 10-15px 区域采样，检查平均明度 V < 阈值
8. 输出最佳候选 ZoneDetection，或 None
```

**quality 计算规则（第一版简化）**：

```python
quality = w(area_score) * w(solidity_score) * w(border_score)

# area_score: 面积占比在合理范围内 → 1.0，太大或太小打折
# solidity_score: 轮廓饱满度，越接近 1.0 越好
# border_score: 黑边验证通过 → 1.0，不通过 → 0.3
```

**HSV 参数（初始值，可在 .env 或 config.py 中调整）**：

```python
# 纯黄色 HSV 范围
YELLOW_H_LOW, YELLOW_H_HIGH = 20, 35
YELLOW_S_LOW, YELLOW_S_HIGH = 80, 255
YELLOW_V_LOW, YELLOW_V_HIGH = 120, 255

# 黑边验证：外扩区域的平均 V 通道值阈值
BORDER_V_THRESHOLD = 80

# 面积门槛
ZONE_MIN_AREA_PX = 800
ZONE_MAX_AREA_RATIO = 0.50  # 占画面最多 50%

# 形状约束（从现有代码继承，可适当放宽）
ZONE_MIN_ASPECT = 1.5
ZONE_MAX_ASPECT = 15.0
ZONE_MIN_SOLIDITY = 0.60
ZONE_MIN_Y_RATIO = 0.30  # 中心点在画面中下部
```

## 阶段 3：前进定点（视觉伺服推进）

### 目标

让狗朝着「区域本身」前进，不是朝着人追，不是按固定时间前进。能根据区域中心和面积动态前进，并在接近区域时稳定停下。

**这一步只做前进闭环，不做返航。**

**改动点 2**：重写 `_on_advancing()`

```python
async def _on_advancing(self, detections, frame):
    # 超时保护
    elapsed = time.monotonic() - self._guard_start_time
    if elapsed >= self._config.GUARD_MAX_DURATION_S:
        logger.warning("[GuardMission] 超时，返航")
        await self._start_returning()
        return

    # 1. 获取当前区域
    zone = self._current_zone  # 由 process_frame 中已检测
    if zone is None:
        self._clear_counter += 1
        if self._clear_counter > 30:
            logger.warning("[GuardMission] 持续丢失区域，返航")
            await self._start_returning()
        else:
            await self._send_command_safe("stop")
        return
    self._clear_counter = 0

    # 2. 调用 servo 计算指令
    cmd, is_arrived = self._servo.compute_advancing(
        curr_bbox=zone.bbox,
        frame_width=self._frame_width,
        frame_height=self._frame_height,
        max_view_ratio=self._config.GUARD_MAX_VIEW_RATIO,
    )

    # 3. 执行
    await self._send_command_safe(cmd)

    # 4. 人是否已离开
    persons = [d for d in detections if d.class_name == "person" and d.confidence >= 0.4]
    still_on = any(self._is_foot_in_zone(p) for p in persons)

    if not still_on and elapsed >= self._config.GUARD_MIN_DURATION_S:
        self._clear_counter += 1
        if self._clear_counter >= self._clear_frames:
            logger.info("[GuardMission] 人已离开，返航")
            await self._start_returning()
    else:
        self._clear_counter = 0
```

## 阶段 4：后退回起点（视觉闭环返航）

### 目标

**改动点 2**：重写 `_on_returning()`

```python
async def _on_returning(self, detections, frame):
    # 返航超时保护（不能无限后退）
    elapsed = time.monotonic() - self._return_start_time
    if elapsed >= self._config.GUARD_RETURN_DURATION_S * 3:
        # 超过正常返航时长 3 倍仍未到位 → 强制停止
        logger.warning("[GuardMission] 返航超时，强制停止")
        await self._send_command_safe("stop")
        await asyncio.sleep(0.5)
        await self._send_command_safe("sit")
        self._finish_mission_and_reset(frame)
        return

    # 1. 获取当前区域
    zone = self._current_zone
    if zone is None:
        # 区域丢失 → 只能盲退（降级保护）
        await self._send_command_safe("backward")
        return

    if self._start_zone is None:
        # 没有起始快照（异常情况）→ 降级为定时后退
        await self._send_command_safe("backward")
        return

    # 2. 调用 servo 计算返航指令
    cmd, is_returned = self._servo.compute_returning(
        curr_bbox=zone.bbox,
        start_bbox=self._start_zone.bbox,
        frame_width=self._frame_width,
        pos_tolerance_px=self._config.GUARD_RETURN_POS_TOLERANCE_PX,
        area_tolerance_ratio=self._config.GUARD_RETURN_AREA_TOLERANCE_RATIO,
    )

    # 3. 执行指令
    await self._send_command_safe(cmd)

    # 4. 返航完成判定（需要连续 N 帧稳定）
    if is_returned:
        self._return_stable_counter += 1
        if self._return_stable_counter >= self._return_stable_threshold:
            logger.info("[GuardMission] 视觉返航完成")
            await self._send_command_safe("stop")
            await asyncio.sleep(0.5)
            await self._send_command_safe("sit")
            self._finish_mission_and_reset(frame)
    else:
        self._return_stable_counter = 0
```

## 阶段 5：遮挡保护 + 丢失保护 + 状态机精简

### 目标

人踩在区域上时，系统不跳区、不瞎走、必要时停机保护。同时清理废弃代码和精简状态机。

### 修改文件

#### `yellow_zone_detector.py`（修改）

新增遮挡容错机制：

```python
class YellowZoneDetector:
    def __init__(self, ...):
        # ...
        self._last_good_zone: Optional[ZoneDetection] = None
        self._lost_frames: int = 0
        self._max_lost_frames: int = 30  # 连续丢失帧数阈值

    def detect(self, frame: bytes) -> Optional[ZoneDetection]:
        raw = self._detect_raw(frame)  # 原始检测管线

        if raw is not None and raw.quality >= 0.7:
            # 高质量检测 → 更新稳定区域
            self._last_good_zone = raw
            self._lost_frames = 0
            return raw
        elif raw is not None and raw.quality >= 0.3:
            # 低质量但有结果 → 沿用 last_good_zone，微修正
            self._lost_frames = 0
            if self._last_good_zone is not None:
                return self._last_good_zone  # 冻结
            return raw
        else:
            # 完全丢失
            self._lost_frames += 1
            if self._lost_frames <= self._max_lost_frames and self._last_good_zone is not None:
                return self._last_good_zone  # 短时容错，沿用历史
            return None  # 真正丢失
    
    @property
    def is_zone_lost(self) -> bool:
        return self._lost_frames > self._max_lost_frames
    
    @property
    def lost_frames(self) -> int:
        return self._lost_frames
```

    return
