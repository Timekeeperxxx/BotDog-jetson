AI 核心逻辑指引 (LLM Implementation Guide)

核心业务流逻辑:

  • 视频闭环：从机载端原始采集到前端 WebRTC 呈现，全链路延迟控制为软件调优核心。
  • 指令闭环：采用 MAVLink 2.0 协议包，实现地对空指令的原子化执行与校验。
  • AI 识别流：属于旁路异步逻辑，检测到异常后触发截帧入库，不阻塞实时视频主路。

软件工程化建议:

  • 后端：重点关注 FastAPI 的 WebSocket 性能，建议对高频遥测数据进行二进制压缩传输。
  • 机载端：ONNX 推理应利用硬件加速核心（NVIDIA TensorRT），降低 CPU 占用率。
